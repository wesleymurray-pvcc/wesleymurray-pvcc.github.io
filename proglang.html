Chapter 6: Programming Languages

Machine Language Sequences of instructions that are encoded in numeric digits.
Debugging Errors in the machine language that must be located and corrected.
Assembler Developed to convert mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system for representing programs.
Translator Translates programs expressed in high-level primitives into machine-language programs.
Compiler Compiles several machine instructions into short sequences to simulate the activity requested by a single high-level primitive.
Interpreter Execute the instructions as they are translated instead of recording the translated version for future use.
Cross-platform Software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executabl throughout a network.
Natural Language Are precisely defined by grammar.
Formal Language Evolved over time without formal grammatical analysis.
Machine Independence Does not rely on characteristics of a particular machine.
High-level Programming Language Provides more abstractions for the programmer and make it easier for people to read and write a program.
Object Oriented Programming (OOP) Software system is viewed as collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects.
Scripting Language Languages used to perform administrative tasks rather than to develop complex programs.
Integer Refers to numeric data consisting of whole numbers.
Float Refers to numeric data that might contain calues other than whole numbers.
String An array data structure of bytes (or words) that stores a sequence of elements, typically characters, using some character encodiing.
Primitive Data Type Data types that are included as primitives in a programming language.
Data Structure The conceptual shape or arrangement of data.
List Used to store multiple items in a single variable, they are created using square brackets[]
Tuple Used to store multiple items in a single variable, written with round brackets()
Set Sets are used to store multiple items in a single variable, they are unordered, unchangeable, and unindexed.
Dictionary Used to store data calues in key: value pairs, a collection is ordered, changeable, and does not allow duplicates.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function that can only be referenced within that function.
Global Variable Variables whose scopes are not restricted to a particular part of a program.
Source Program During process of translation the source program is the original pre-translated form.
Object Program The translated version of the source program.
