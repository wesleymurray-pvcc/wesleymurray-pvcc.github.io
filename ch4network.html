Operating Systems Terminology

    Network Classification

  • LAN: Local Area Network, consists of a collection of computers in a single building or building complex
  • MAN: Metropolitan Area Network, a network of intermediate such as one spanning a local community
  • WAN: Wide Area Network, links machines over greater distances between cities, and countries etc.

    Network Models

  • Bus Network: Common communication line called bus connects machines
  • Star Network: One machine serves as central focus point to which all other machines are connected
