Operating Systems Terminology

    Network Classification

  • LAN: Local Area Network, consists of a collection of computers in a single building or building complex.
  • MAN: Metropolitan Area Network, a network of intermediate such as one spanning a local community.
  • WAN: Wide Area Network, links machines over greater distances between cities, and countries etc.

    Network Models

  • Bus Network: Common communication line called bus connects machines.
  • Star Network: One machine serves as central focus point to which all other machines are connected.

    Protocols

  • CSMA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection. Right to transmit message is governed by protocols.
  • CSMA/CA Wifi: Carrier Sense, Multiple Access with Collision Avoidance. Trying to avoid data collision rather than detect it.

    Combining Networks

  • repeater: Device that passes signals back and forth between the two original buses.
  • bridge: Like a repeater but more complex, it seeks a destination and will not send message across unless it has defined destination.
  • switch: Bridge with multiple connection allowing it to connect several buses rather than just two.
  • router: Connect to form an internet, special purpose computers used for forwarding messages.
  • gateway: Point at which one network is linked to the internet.

    p2p: peer-to-peer

  • distributed systems: Software units that execute as processes on different computers.
  • cloud computing: Huge pools of shared computers on the network that can be allocated for use by clients as needed.

    Internet Addressing

  • Tier 1: High speed, high capactity, international WANs considered the backbone of the internet.
  • Tier 2: More regional in scope and less potent in their capablilies.
  • Tier 3: Independent, intranets, generally run by a cable of phone company.

    Ip Address

  • ICANN: Internet Corporation for Assigned Names and Numbers, non-profit corporation established to coordinate the Interet's operation.
  • ICANN
  • DNS: Internet wide directory system based around domain names.
  • TLD: Top-level domain, domain names charcterized by suffixes.
  • Country-code TLDs: Domain names for countries.
  • Country-code Domain Names

    Internet Protocols

  • FTP: File Transfer Protocol, application for accessing another computer from a great distance using the Telnet protocol.
  • HTTP: Hyper Text Transfer Protocol, a more sophisticate file transfer protocol generally in use today.
  • SMTP: Simple Mail Transfer Protocol, defines a way that two computers on the network may interact when transmitting an email message from one host to the other.

    World Wide Web

  • URL: Uniform Resource Locator, unique address given to a document on the internet so it may be retrieved.
  • client-side activities: Activities performed by the client like filling out a form.
  • server-side activities: Activities performed by the server.

    Internet four-layer software

  • application layer: Consists of those software units such as clients and servers that use Internet communication to carry out their tasks.
  • transport layer: Used by application layer to send and receive messages over the Internet like a shipping company sends packages.
  • network layer: Decides which directiion a packet should be sent at each step along the packet's path through the Internet.
  • link layer: Has the responsibility to transfer the packet.

    Internet is:

  • packet switching network: The Internet in which digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data.
  • fault tolerant: Refers to the system's ability to continue operating deespite failure or malfunctions.
  • redundant: Process of providing multiple paths for traffic so that data can keep flowing even in the event of a failure.

    Cybersecurity

  • malware: Malicious software that attacks a computer system and its contents often to steal information or disrupt essential functions.
  • virus: Software that infects a computer by inserting itself into programs that already reside in the machine.
  • worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • Trojan horse: A program that enters a computer system disguised as a desirable program, such as a game or useful utility package that is willingly imported by the victim.
  • spyware: Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • phishing: Technique of obtain information explicitly by simply asking for it, new version of the telephone con.
  • DDos attack: Distributed denial of service attack is an attack generated by generating a large number of messages in a short period of time to overwhelm the servers, these attacks are generated from numerous unsuspecting computers.

    Protection

  • firewall: Traffic is filtered through this point to prevent infiltration.
  • spam filter: Firewalls designed to block unwanted emails.
  • proxy server: Software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • antivirus software: Used to detect and remove the presence of known viruses and other infections.
  • cryptography: Study of sending and receiving messages ercurely in the presence of adversaries.
  • public key encryption: Involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
