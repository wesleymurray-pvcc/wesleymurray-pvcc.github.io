Operating Systems Terminology

TERM DEFINITION
Operating System The software that controls the overal operation of the computer including file management, the user interface, and the program execution environment.
Job Queue In batch processing systems, the jobs residing in mass storage wait for execution here in a FIFO structure.
FIFO First-in, first-out: objects are removed from the queue in the order which they arrived.
Interactive Processing Allows the program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing Operating systems that provide service to multiple users at the same time.
Multiprogramming When time is divided into intervals and the the execution of each job is restricted to only one interval at a time.
Application Software Consists of the programs for performing tasks particular to the machine's utilization.
System Software Performs those tasks that are common to computer systems in general.
Utility Software Programs for performing activities that are fundamental to computer installations but not included in the operating system.
Graphical User Interface In which objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell Older user interfaces that communicate with users through textual messages using a keyboard and monitor screen.
Kernel Contains those software components that perform the very basic functions required by the computer installation.
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager Coordinates the use of the machine's mass storage facilities
Memory Manager Tasked with coordinating the machine's use of main memory.
Virtual Memory The large "fictional" memory space created by paging.
Boot Strapping Procedure that transfers the operating system from mass storage into main memory.
ROM Read-only memory is memory whose contents can be read but not altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is wating for a resource that is allocated to another.
Spooling A technique of holding data for output at a later but more convenient time.
Super User or Administrator Person with highly priviliged access to the operatring system established at login that can administer the system functions.
Multi Factor Authentication Utilizing a password plus a second or third identification question or factor like a smartphone link.
